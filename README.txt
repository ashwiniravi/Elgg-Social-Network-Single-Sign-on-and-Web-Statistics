Integrated Single Sign On and Web statistics to the existing elgg social network site.

Elgg-Social-Network-Single-Sign-on-and-Web-Statistics/account/fbconnect.php  : Integrated fbConnect Sinle Sign On

Elgg-Social-Network-Single-Sign-on-and-Web-Statistics/account/googleplus_new.html : Added GooglePlus Sinle Sign On

Elgg-Social-Network-Single-Sign-on-and-Web-Statistics/account/stats_comment_graph.php : Displays graph statistics for the most commented blogs

Elgg-Social-Network-Single-Sign-on-and-Web-Statistics/account/stats_friends_graph.php : Displays graph statistics (day,weekly,monthly) for each friend request accepted 

Elgg-Social-Network-Single-Sign-on-and-Web-Statistics/account/stats_inbox_graph.php : Displays graph statistics (day,weekly,monthly) for each messages received

Elgg-Social-Network-Single-Sign-on-and-Web-Statistics/account/stats_sentItems_graph.php : Displays graph statistics (day,weekly,monthly) for each message sent.

Elgg-Social-Network-Single-Sign-on-and-Web-Statistics/account/stats_visitors_graph.php : Displays graph statistics (day,weekly,monthly) based on the number of vistors visited the user profile.
